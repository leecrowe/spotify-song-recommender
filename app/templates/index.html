This model was built using the KNearest Neighbors algorithm. It takes songs directly from Spotify before running them through a TF-IDF model to transform to vector before inputting them into the KNN model to find top 5 similar songs.

About

I currently am attending Lambda School, an intensive online data science course for a career in applied statistics and predictive modeling. Building real, ready-to-ship products with a team of other students generating real work experience is a key feature of Lambda School. This app is a product of that feature.

After gathering song data from Spotify a predictive model was built and trained. This model allows a user to input a song they like and get find the top recomendations for that song. The top recommended songs can be used to find a new favorite song or even help you reconnect with a past favorite.
